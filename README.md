## Basic Usage

Run in your terminal for root directory of project

```shell
sudo chmod +x build.sh run.sh stop.sh
```

## Build image

```shell
./build.sh
```

## Run System

```shell
## start container
./run.sh ## para startar e subir o server
## stop container
./stop.sh ## para parar os containers
```
